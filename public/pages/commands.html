Commands for Twitch chatters

Here are all the currently available commands for If you would like to see more commands, please let me know!

!current - displays what the streamer is currently playing and for how long, eg. In game as Jax for 19m
  • runes - optional, displays the streamer's runes for the current game
  • bans - optional, displays the bans for the streamer's current game
!last - displays stats from the streamer's last game, eg. Jake went 0/10/0 on Singed Top and won [5h30m ago, account: jakehoffmann (NA), <matchhistorylink>] !rank - displays the league, division, and LP of all the streamer's registered summoners

The following commands are only for the streamer

!title - changes the base of your title only. This will not affect the current game info at the front, if any. You will want to disable !title commands from other Twitch chat bots so they don't conflict with each other.
